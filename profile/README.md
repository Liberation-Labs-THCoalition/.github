## The Problem Is Simple

Corporations and governments have AI. The people don't. Every day that gap widens, power consolidates further. Surveillance tools get smarter. Public records get harder to access. Corruption gets harder to trace.

**We're closing that gap.** Liberation Labs builds free, open-source AI tools — digital volunteers on demand — so that the communities doing the hardest work don't get left behind.

## 🧭 Our Principles
